# 🛒 ShopMaster — FAQ (Часто задаваемые вопросы)

## 📱 Для покупателей

### Как начать покупать?
1. Найдите бота в Telegram или перейдите по ссылке
2. Нажмите **"Начать"** или отправьте `/start`
3. Выберите **"🛍 Каталог"** в меню
4. Добавьте товары в корзину
5. Перейдите в **"🛒 Корзина"** и оформите заказ

### Как применить промокод?
При оформлении заказа (шаг 1/3) бот спросит промокод. Введите код или нажмите "Пропустить".

**Примеры рабочих промокодов:**
- `SALE2024` — скидка 20% (если админ создал)
- `WELCOME10` — скидка 10% для новых клиентов

### Как оплатить заказ?
После подтверждения заказа появится кнопка **"Оплатить"**. Нажмите её и следуйте инструкциям Telegram Payments. Принимаются карты Visa, Mastercard, МИР.

### Как отследить заказ?
Нажмите **"📦 Мои заказы"** в меню. Там виден статус каждого заказа:
- ⏳ **Ожидает** — заказ создан, ждёт оплаты
- 📦 **В обработке** — оплачен, собирается
- 🚚 **Отправлен** — передан в доставку
- ✅ **Доставлен** — заказ у вас!

### Как связаться с поддержкой?
Нажмите **"📞 Поддержка"** в меню или напишите @support_manager. Работаем с 9:00 до 21:00 МСК.

### Как сменить язык?
Отправьте команду `/lang` и выберите: 🇷🇺 Русский, 🇬🇧 English, 🇺🇿 O'zbekcha

## 🔧 Для администраторов

### Как создать промокод?
**В Telegram боте** (только админ):
```
/promo_create КОД ТИП ЗНАЧЕНИЕ ЛИМИТ [МИН_СУММА] [МАКС_СКИДКА]
```

**Примеры:**
```
/promo_create SALE2024 percent 20 100
```
→ Скидка 20%, 100 использований, без минимальной суммы

```
/promo_create MINUS500 fixed 500 50 3000
```
→ Скидка 500₽, 50 использований, минимальная сумма 3000₽

```
/promo_create BLACKFRIDAY percent 30 500 5000 2000
```
→ Скидка 30%, 500 использований, мин. сумма 5000₽, макс. скидка 2000₽

**Типы скидок:**
- `percent` — процент от суммы заказа
- `fixed` — фиксированная сумма в ₽

### Как посмотреть список промокодов?
Отправьте `/promo_list` — бот покажет все активные и неактивные промокоды с статистикой использования.

### Как деактивировать промокод?
Через CRM: откройте `https://your-domain.onrender.com/crm`, найдите промокод и нажмите "❌ Деактивировать".

### Как экспортировать заказы?
**Вариант 1 — через CRM:**
1. Откройте `https://your-domain.onrender.com/crm`
2. Нажмите **"📊 Excel"** или **"📄 CSV"**
3. Файл автоматически скачается

**Вариант 2 — через бот:**
- `/export_excel` — скачать Excel
- `/export_csv` — скачать CSV

### Как изменить статус заказа?
1. Откройте CRM: `https://your-domain.onrender.com/crm`
2. Найдите заказ
3. Нажмите кнопку действия:
   - ✅ **В обработку** (для новых)
   - 🚚 **Отправить** (для собранных)
   - ✅ **Доставлен** (для отправленных)
4. Клиент автоматически получит уведомление в Telegram

### Как синхронизировать заявки из Telegram?
В CRM нажмите кнопку **🔄** (справа вверху) или **✈️** (FAB-кнопка справа внизу). Заявки из бота появятся в CRM за 1-2 секунды.

### Как посмотреть статистику?
Отправьте `/stats` — бот покажет:
- Новые заказы за сегодня
- Заказы в обработке
- Выручку за месяц
- Топ товаров

## 🔧 Технические вопросы

### Почему 404 на /docs, /crm, /miniapp?
**Причина:** Сервис не запущен или пути не настроены.

**Решение:**
1. Проверьте, что `DEBUG=true` в ENV (для /docs)
2. Убедитесь, что папки `miniapp/` и `crm/` существуют в проекте
3. Перезапустите сервис на Render

### Почему бот не отвечает?
1. Проверьте `BOT_TOKEN` в ENV
2. Убедитесь, что Worker (bot) запущен на Render
3. Проверьте логи: Render Dashboard → Logs

### Как добавить новый товар?
Через API (требуется admin JWT):
```bash
curl -X POST https://your-domain.onrender.com/products/   -H "Authorization: Bearer YOUR_TOKEN"   -H "Content-Type: application/json"   -d '{
    "name": "Новый товар",
    "description": "Описание",
    "price": 99.99,
    "stock": 100,
    "image_url": "https://example.com/image.jpg"
  }'
```

### Как сделать бэкап базы данных?
Render автоматически делает бэкапы PostgreSQL. Для ручного бэкапа:
```bash
pg_dump $DATABASE_URL > backup.sql
```

## 💰 Платежи

### Как подключить оплату?
1. В @BotFather: `/mybots` → **Payments** → Выберите провайдера (ЮKassa/Stripe/SberPay)
2. Получите `PAYMENT_PROVIDER_TOKEN`
3. Добавьте в ENV на Render

### Почему оплата не проходит?
- Проверьте `PAYMENT_PROVIDER_TOKEN`
- Убедитесь, что сумма заказа > 0
- Для тестов используйте тестовые карты провайдера
